HTML

HTML is the standard language for webpages.
It stands for Hypertext Markup Language
With HTML you can create your own Website.



Below is a simple example HTML page.

Explanation of the example

  • <!DOCTYPE html> declares that the document is a HTML5 document
  • <html> is where the where everything else goes in
  • <head> is where all the elements that aren't displayed on the page go such as the <title> & <stlye> tags
  • <title> defines the title of the tab displayed in the browser window
  • <body> is where the elements that are physicaly displayed on the page go
  • <h1> displays a large header
  • <p> displays a paragraph
